# Hacktoberfest 2022
## Even odd in C++

### Problem Statement
Write a program to find the even odd numbers in C++.

### Input Format
Accept a number n as input.

### Output Format
Complete the function which returns the number is true else it returns false.

### Sample Input
```
5
```

### Sample Output
```
Number is Odd
```
