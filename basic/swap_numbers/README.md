# Hacktoberfest 2022
## Swap two numbers in C++

### Problem Statement
Write a program to swap the two numbers in C++.

### Input Format
Take two numbers a and b as input.

### Output Format
Complete the function which swap the two numbers.

### Sample Input
```
5 6
```

### Sample Output
```
6 5
```
