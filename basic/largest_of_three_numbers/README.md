# Hacktoberfest 2022
## largest of three numbers  in C++

### Problem Statement
Write a program to find the largest number in C++.

### Input Format
Accept 3 numbers a,b,c as input.

### Output Format
Complete the function which returns the true if the given number is largest among 3 numbers.

### Sample Input
```
5,6,9
```

### Sample Output
```
9
```
